$$.toast(options)

概述

Toast用于临时显示某些信息，和Dialog不一样的是，Toast是没有焦点的，而且Toast显示的时间有限，会在数秒后自动消失。Toast主要用于向用户显示提示消息，这些信息通常是轻量级操作的成功、失败或等待状态信息。

  • options.title - string 信息标题
  • options.content - string 信息内容，可选。
  • options.site - number | string 设置位置，顺序[上 右 下 左] 如：[100,'10%',10,10] 或者 ['10%',,10] 可选。
  • options.time - number 信息显示时间，可选。
  • options.className - string 自定义样式名，可选。

实例

弹层:

HTML 代码:
<button type="button" class="Btn BtnPrimary InkRipple ToastBtn">默认Toast样式</button>
js 代码:
$$(".ToastBtn").on('click',function(){
    $$.toast({
        text:'默认Toast样式'
    });
});
