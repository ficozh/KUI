$$.imgReady(url, ready, load, error)

概述

加载图片时，我们需要获得图片是否已经加载完毕，进行后续的事件处理。

它可以更快的获取图片的宽和高尺寸。

  • url - string 需要处理的图片地址
  • ready - function 图片 ready 后执行的回调函数，图片 ready 时，图片还没有在页面上渲染。
  • load - function 图片 load 后执行的回调函数，图片 load 时，图片已经在页面上渲染。
  • error - function 图片加载出错时的回调函数。

js 方法

$$.imgReady('../images/02.jpg',function(){ 
    //ready
},
function(){ 
    //load
},
function(){ 
    //error
})
