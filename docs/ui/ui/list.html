list(列表)

概述

列表是种多功能且强大的交互组件。列表将数据展现在一个可滚动的多行的列表中，可以将数据划分成不同的片段/组

列表有以下用途：

  • 让用户可以浏览结构化的多级数据
  • 展现一个索引列表
  • 在可视化的不同分组中，显示详细信息和控件
  • 展现一个可以选择的列表

实例

列表:

<ul class="ListBlock">
    <li class="ListItem">
        <div class="ItemCon">
            <div class="ItemMedia">
                <img class="Icon" src="../../../images/svg/iconfont-user.svg" />
            </div>
            <div class="ItemInner">
                <div class="ItemTitle"></div>
                <div class="ItemAfter"></div>
            </div>
        </div>
    </li>
</ul>

其中：

  • ItemCon - ItemMedia 和 ItemInner 的主要弹性(flex)容器。必选元素。
  • ItemMedia - 媒体元素（如图标，图片等）的容器。可选元素。
  • ItemInner - ItemTitle和 ItemAfter 的主要弹性(flex)容器。必选元素。
  • ItemTitle - 单行列表项目标题。必选元素。
  • ItemAfter - 列表项目标签。可以包含任意额外的html元素（例如标签文本、徽章、switch/toggle、按钮等等）。可选元素。

展示效果:

  • Ivan Petrov
    CEO
  • John Doe
    13
  • Jenna Smith

列表分割元素

列表分隔元素是一个简单的列表元素并包含标题，它在视觉上分隔两个列表元素：

<li class="ItemDivider">列表分割</li>

展示效果:

  • John Doe
    13
  • 列表分割
  • Jenna Smith

分组列表

有时我们需要将列表区里的元素分组，在这种情况下，我们需要用额外的分组元素：

<div class="ListBlock">
    <div class="ListGroup">
        <ul>
            <li class="ListItem GroupTitle">标题A</li>
            <li class="ListItem">
                <div class="ItemCon InkRipple">
                    <div class="ItemInner">
                        <div class="ItemTitle">Aaron</div>
                    </div>
                </div>
            </li>
            <li class="ListItem">
                <div class="ItemCon InkRipple">
                    <div class="ItemInner">
                        <div class="ItemTitle">Adam</div>
                    </div>
                </div>
            </li>
        </ul>
    </div>
</div>

展示效果:

  • 标题A
  • Aaron
  • Abbie
  • Adam
  • 标题B
  • Bailey
  • Barclay
  • 标题C
  • Caiden
  • Candy

内嵌列表区域:

列表区还有内嵌（非全屏）样式，只需要给列表区域元素加上“Inset”类

<div class="ListBlock Inset">

圆角块级列表区域:

增加“ListCard”可以使列表以圆角块级展示

<div class="ListBlock ListCard">

自适应列表区域:

平板电脑适用

<div class="ListBlock ListTabletInset">
