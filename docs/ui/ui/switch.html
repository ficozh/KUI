switch(开关)

概述

流行的按钮开关形式。最主要的是用CSS来控制显示的效果。

实例

开关:

<label class="LabelSwitch">
    <input type="checkbox" />
    <div class="checkbox"></div>
</label>
<label class="LabelSwitch LabelSwitchText">
    <input type="checkbox" />
    <div class="checkbox"></div>
</label>
