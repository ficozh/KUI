设置语言

概述

增加修改默认语言包方法。

加载库之前需要注入 kelatlocale 对象到 window 对象中，例如：

js 代码:
<script type="text/javascript">
  window.kelatlocale={
      "SYSTEM_LANGUAGE": {
      "MODAL": {//弹出框
        TIPS:"提示",
        OK:"确定",
        CANCEL:"取消"
      },
      "IMG":{//图片信息
        IMGTIPS:"图片格式不正确或者跨域请求！",
        IMGPlaceholder :'data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7'
      },
      //加载
      "LOADING": [
        '正在加载...'
      ]
      },
      //全局作用域id
      "WrapperArea": 'WrapperArea'
  };
</script>
<script type="text/javascript" src="../script/kelat.js"></script>
