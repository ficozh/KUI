$$.imgRender(options, callback)

概述

对图片进行大小调整，输出 base64 格式图片。

它可以在本地对图片进行处理后上传，提升上传速度，节省流量。

  • options - Array path 图片路径，width，图片生成后的宽度
  • callback - function 返回新的图片 base64 信息。

注意：图片渲染需要浏览器支持 canvas 。

js 方法

$$.imgRender({
    path:'../images/01.jpg',
    width:600,
},function(newImg){ 
    
})
