$$.prompt(text,time)

概述

我们需要调用以下任一一种方法来打开弹层:

$$.prompt(text)

$$.prompt(text,time)

  • text - string 文本
  • time - number 显示的时间，单位毫秒 ，默认2000毫秒，可选。

实例

弹层:

HTML 代码:
<button type="button" class="Btn BtnPrimary InkRipple PromptBtn">自动消失提示框</button>
js 代码:
$$(".PromptBtn").on("click",function(){
  $$.prompt("欢迎体验KMUI")
});
