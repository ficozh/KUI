返回值:Object$$.getUrlParams([url,type])

概述

将连接参数序列化成为对象。

参数

urlString

将指定的url地址进行序列化。

typeObject

指定特殊字符截取。需要设置两个值，[value0,value1] , value0是需要从哪个特殊字符截取，value1是需要截取的 key 。

示例

描述:

获取链接参数

js 代码:
$$.getUrlParams();

描述:

获取指定链接参数

js 代码:
var UrlSet = $$.getUrlParams({'url':'http://www.kui.com/kui.html?id=1'});
结果:
UrlSet.id  // 1

描述:

获取指定参数的值。没有url参数时， 获取当前地址栏中的url，

js 代码:
var UrlSet = $$.getUrlParams({'url':'http://www.kui.com/kui.html?id=1&name=name','type':['name=','name']});
结果:
UrlSet.name  // name
UrlSet.id  // undefined
