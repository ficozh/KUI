返回值:kelatremoveData([name|list])

概述

在元素上移除存放的数据

与$$(...).data(name, value)函数作用相反

参数

[name]String

存储的数据名

[list]Array,String

移除数组或以空格分开的字符串

示例

描述:

从元素中删除之前添加的数据：

kelat 代码:
$$("#btn2").click(function(){
  $$("div").removeData("greeting");
  alert("Greeting is: " + $$("div").data("greeting"));
});
