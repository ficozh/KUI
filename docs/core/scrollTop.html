返回值:IntegerscrollTop(position, duration,fn)

概述

获取匹配元素相对滚动条顶部的偏移。

参数

positionString, Number

设定垂直滚动条值

durationNumber

在指定的时间（duration 毫秒)滚动到到指定的位置

示例

无参数描述:

获取第一段相对滚动条顶部的偏移

HTML 代码:
<p>Hello</p><p>2nd Paragraph</p>
js 代码:
var p = $$("p:first");
$$("p:last").text( "scrollTop:" + p.scrollTop() );
结果:
<p>Hello</p><p>scrollTop: 0</p>

参数val 描述:

设置相对滚动条顶部的偏移

js 代码:
$$("div.demo").scrollTop(300);
