返回值:kelatprepend(content)

概述

向每个匹配的元素内部前置内容。

这是向所有匹配元素内部的开始处插入内容的最佳方式。

参数

contentString, Element

要插入到目标元素内部前端的内容

示例

描述:

向所有段落中前置一些HTML标记代码。

HTML 代码:
<p>I would like to say: </p>
js 代码:
$$("p").prepend("<b>Hello</b>");
结果:
[ <p><b>Hello</b>I would like to say: </p> ]

描述:

将一个DOM元素前置入所有段落

HTML 代码:
<p>I would like to say: </p>
<p>I would like to say: </p>
<b class="foo">Hello</b>
<b class="foo">Good Bye</b>
js 代码:
$$("p").prepend( $$(".foo")[0] );
结果:
<p><b class="foo">Hello</b>I would like to say: </p>
<p><b class="foo">Hello</b>I would like to say: </p>
<b class="foo">Good Bye</b>

描述:

向所有段落中前置一个kelat对象(类似于一个DOM元素数组)。

HTML 代码:
<p>I would like to say: </p><b>Hello</b>
js 代码:
$$("p").prepend( $$("b") );
结果:
<p><b>Hello</b>I would like to say: </p>
