返回值:kelatprependTo(content)

概述

把所有匹配的元素前置到另一个、指定的元素元素集合中。

实际上，使用这个方法是颠倒了常规的$(A).prepend(B)的操作，即不是把B前置到A中，而是把A前置到B中。

参数

contentString

用于匹配元素的表达式

示例

描述:

把所有段落追加到ID值为foo的元素中。

HTML 代码:
<p>I would like to say: </p><div id="foo"></div>
js 代码:
$$("p").prependTo("#foo");
结果:
<div id="foo"><p>I would like to say: </p></div>
