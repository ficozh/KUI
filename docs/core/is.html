返回值:Booleanis(selector)

概述

根据选择器、DOM元素或 kelat 对象来检测匹配元素集合，如果其中至少有一个元素符合这个给定的表达式就返回true。

如果没有元素符合，或者表达式无效，都返回'false'。

参数

selectorString,object

包含供匹配当前元素集合的选择器表达式、DOM元素、kelat对象、window对象。

示例

参数expr 描述:

由于input元素的父元素是一个表单元素，所以返回true。

HTML 代码:
<form><input type="checkbox" /></form>
js 代码:
$$("input[type='checkbox']").parent().is("form")
结果:
true

回调函数 描述:

判断点击li标签增加背景色为红色，如果点击的是第2个strong,当前的li增加背景色为绿色，

HTML 代码:
<ul>
  <li><strong>list</strong> item 1 - one strong tag</li>
  <li><strong>list</strong> item <strong>2</strong> - two <span>strong tags</span></li>
  <li>list item 3</li>
</ul>
js 代码:
$$("li").click(function() {
  var $li = $$(this),
    isWithTwo = $li.is(function() {
      return $$('strong', this).length === 2;
    });
  if ( isWithTwo ) {
    $li.css("background-color", "green");
  } else {
    $li.css("background-color", "red");
  }
});
