返回值:Boolean$$.isNumber(value)

概述

确定它的参数是否是一个数字。

$.isNumber() 方法检查它的参数是否代表一个数值。如果是这样，它返回 true。否则，它返回false。该参数可以是任何类型的

参数

value

用于测试的值。

示例

描述:

检测是否为数字：

js 代码:
$.isNumber("-10");  // true
$.isNumber(16);     // true
$.isNumber(0xFF);   // true
$.isNumber("0xFF"); // true
$.isNumber("8e5");  // true (exponential notation string)
$.isNumber(3.1415); // true
$.isNumber(+10);    // true
$.isNumber(0144);   // true (octal integer literal)
$.isNumber("");     // false
$.isNumber({});     // false (empty object)
$.isNumber(NaN);    // false
$.isNumber(null);   // false
$.isNumber(true);   // false
$.isNumber(Infinity); // false
$.isNumber(undefined); // false
