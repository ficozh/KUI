返回值:kelatfind(expr|obj|ele)

概述

搜索所有与指定表达式匹配的元素。这个函数是找出正在处理的元素的后代元素的好方法。

所有搜索都依靠jQuery表达式来完成。这个表达式可以使用CSS1-3的选择器语法来写。

参数

exprString

用于查找的表达式

kelat object object

一个用于匹配元素的kelat对象

elementDOMElement

一个DOM元素

示例

描述:

从所有的段落开始，进一步搜索下面的span元素。与$$("p span")相同。

HTML 代码:
<p><span>Hello</span>, how are you?</p>
js 代码:
$$("p").find("span")
结果:
[ <span>Hello</span> ]
