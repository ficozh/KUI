返回值:Boolean$$.isWindow(obj)

概述

测试对象是否是窗口（有可能是Frame）。

参数

objObject

用于测试是否为窗口的对象

示例

描述:

测试是否为窗口

js 代码:

<!DOCTYPE html>
<html>
<head>
  <script src="../script/kelat.js"></script>
</head>
<body>
  Is 'window' a window? <b></b>
<script>$$("b").append( "" + $$.isWindow(window) );</script>

</body>
</html>
