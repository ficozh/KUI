返回值:kelat$$.fn.extend(object)

概述

扩展 kelat 元素集来提供新的方法（通常用来制作插件）。

参数

objectObject

用来扩充 kelat 对象。

示例

描述:

增加两个插件方法。

js 代码:
$$.kelatDom.fn.extend({
  check: function() {
    return this.each(function() { this.checked = true; });
  },
  uncheck: function() {
    return this.each(function() { this.checked = false; });
  }
});
结果:
$$("input[type=checkbox]").check();
$$("input[type=radio]").uncheck();
