返回值:Stringcss(name|pro|[,val|fn])

概述

访问匹配元素的样式属性。

参数

nameString

要访问的属性名称

nameArray

一个或多个CSS属性组成的一个数组

propertiesMap

要设置为样式属性的名/值对

name,valueString, Number

属性名,属性值

name,function(index, value)String,Function

1:属性名

2:此函数返回要设置的属性值。接受两个参数，index为元素在对象集合中的索引位置，value是原先的属性值。

示例

参数name 描述:

取得第一个段落的color样式属性的值。

js 代码:
$$("p").css("color");

参数properties 描述:

将所有段落的字体颜色设为红色并且背景为蓝色。

js 代码:
$$("p").css({ "color": "#ff0011", "background": "blue" });

参数name,value 描述:

将所有段落字体设为红色

js 代码:
$$("p").css("color","red");

参数name,回调函数 描述:

逐渐增加div的大小

js 代码:
  $$("div").click(function() {
    $$(this).css({
      width: function(index, value) {
        return parseFloat(value) * 1.2;
      }, 
      height: function(index, value) {
        return parseFloat(value) * 1.2;
      }
    });
  });
