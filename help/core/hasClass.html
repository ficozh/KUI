返回值:BooleanhasClass(class)

概述

检查当前的元素是否含有某个特定的类，如果有，则返回true。

这其实就是 is("." + class)。

参数

classString

用于匹配的类名

示例

描述:

给包含有某个类的元素添加一个class。

HTML 代码:
<div class="protected"></div><div></div>
js 代码:
$$("div").click(function(){
  if ( $$(this).hasClass("protected") )
    $$(this).addClass('active');
});
