返回值:kelatkelat([selector,[context]])

概述

这个函数接收一个包含 CSS 选择器的字符串，然后用这个字符串去匹配一组元素。

kelat 的核心功能都是通过这个函数实现的。 kelat中的一切都基于这个函数，或者说都是在以某种方式使用这个函数。这个函数最基本的用法就是向它传递一个表达式（通常由 CSS 选择器组成），然后根据这个表达式来查找所有匹配的元素。

默认情况下, 如果没有指定context参数，$$()将在当前的 HTML document中查找 DOM 元素；如果指定了 context 参数，如一个 DOM 元素集或 kelat 对象，那就会在这个 context 中查找。

参考文档中 选择器 部分获取更多用于 expression 参数的 CSS 语法的信息。

参数

selector,[context]String,Element,/kelat

selector:用来查找的字符串

context:作为待查找的 DOM 元素集、文档或 kelat 对象。

elementElement

一个用于封装成KDOM元素

objectobject

一个用于封装成kelat对象

elementArrayElement

一个用于封装成KDOM元素数组。

kelat objectobject

一个用于克隆的kelat对象。

kelat()

返回一个空的kelat对象。

示例

描述:

找到所有 p 元素，并且这些元素都必须是 div 元素的子元素。

代码:
$$("div > p");

描述:

设置页面背景色。

代码:
$$(document.body).css( "background", "black" );

描述:

隐藏一个表单中所有元素。

代码:
$$(myForm.elements).hide()

描述:

在文档的第一个表单中，查找所有的单选按钮(即: type 值为 radio 的 input 元素)。

代码:
$$("input:radio", document.forms[0]);

描述:

在一个由 AJAX 返回的 XML 文档中，查找所有的 div 元素。

代码:
$$("div", xml.responseXML);
