$$.actions(options)

概述

操作表是一个上滑面板，用来向用户展示进行一个任务所需的选项。

你也可以使用它来提示用户，确认潜在的危险操作。

操作表包含一个可选的标题和至少一个按钮，每个按钮都对应一个操作。

注意，不建议在大屏幕（iPad）上使用操作表。在大屏幕上，你应该使用弹出框。

操作表是动态元素，只能使用Javascript来创建并弹出它。让我们看一下创建操作表相关的方法:

$$.actions(groups) - 创建并弹出操作表，其包含指定数量群组的按钮

$$.actions(buttons) - 创建并弹出操作表，其包含一个群组，群组包含指定数量的按钮

  • groups - array 一组群组，每个群组包含若干按钮
  • buttons - array 一组按钮，这种情况下，只有一个群组。

这个方法返回动态创建的操作表HTML元素

每个在buttons中的按钮应该作为一个对象，其包含如下参数：

参数 类型 默认值 描述
text String 按钮上的文本（可以是HTML字符串）
bold Boolean false 可选，若为真，则按钮上文本加粗
color String 可选，按钮颜色，待选颜色 参照文本给出的颜色
bg String 可选，按钮颜色，待选颜色 参照背景给出的颜色
label Boolean true 可选，若为真，它会成为标题，而不是按钮
disabled Boolean false 可选，若为真，使按钮禁用
onClick Function 可选，当用户点击该按钮时，会调用这个函数

实例

弹层:

HTML 代码:
<button type="button" class="Btn BtnPrimary InkRipple MenuBtn">底部菜单</button>
js 代码:
$$(".MenuBtn").on("click",function(){
	var actionSheetButtons = [
    [{
      text: '这里可以描述下面的操作这里可以描述下面的操作这里可以描述下面的操作这里可以描述下面的操作',
      label: true
    },{
      text: 'alert警告消息框',
      onClick: function () {
        $$.alert('alert警告消息框!');
      }
    },{
      text: '红色按钮',
      color: 'CR',
      onClick: function () {
        $$.alert('你点击的是红色按钮!');
      }
    }],
    [
      {
        text: '取消',
        bg: 'BBlue'
      }
    ]
  ];
  $$.actions(actionSheetButtons);
});
