loading(加载)

概述

KUI提供了一个好用的加载指示符。 这个加载指示符使用SVG绘制，并使用CSS来进行动画，可以很方便的改变它的大小。

加载指示符有2个颜色供选择：其中一个颜色是配合亮色背景，另一个颜色配合暗色背景，默认的是亮色背景。使用加载提示符非常简单，创建任意标签并加上” Loading ”类。

<div class="Row">
  <div class="Col25 TC">
    <div>默认</div>
    <span class="Loading"></span>
  </div>
  <div class="Col25 TC" style="background-color:#222;color:#fff">
    <div>白色</div>
    <span class="Loading LoadingWhite"></span>
  </div>
  <div class="Col25 TC">
    <div>大号</div>
    <span class="Loading LoadingBig"></span>
  </div>
  <div class="Col25 TC" style="background-color:#222;color:#fff">
    <div>白色</div>
    <span class="Loading LoadingWhite LoadingBig"></span>
  </div>
</div>
  • Loading - 加载指示符
  • LoadingWhite - 白色的加载指示符
  • LoadingBig - 较大尺寸的加载指示符

如果你需要的是一个 Modal 的加载指示，可以使用 JS 动态创建

预加载 Modal

预加载 Modal 用来提示一些后台活动（像Ajax请求）和阻止在这个活动期间的任何用户操作。 打开预加载 modal 我们也需要调用适当方法:

$$.loading(title) - 显示预加载 Modal

  • title - 预加载 Modal 的标题，可选。

$$.unLoading() - 隐藏/关闭 预加载 modal. 因为 预加载 modal 没有任何按钮, 它应该用 JavaScript 来关闭

指示器(indicator)

如果你不需要用像预加载Modal这样如此大的modal 窗口去指示活动, 你可以使用简单并且小的指示器:

$$.indicator() - 显示指示器 Modal

$$.unIndicator() - 隐藏/关闭指示器 modal. 因为指示器modal没有任何按钮, 它需要用JavaScript来关闭
