$$.addNotify(parameters)

概述

使用通知组件，你可以让消息看起来像推送通知一样。

通知API

$$.addNotify(parameters) - 通过指定的参数来创建/显示通知

  • parameters - 包含通知相关的参数的对象。必选。方法返回一个动态创建的通知体的HTML元素

$$.closeNotify(Element) - 关闭指定的通知

  • Element - 通知元素的HTML元素或CSS选择器。必选。

创建一个通知所需要的参数

事件名称 类型 默认值 描述
title string 标题。
subtitle string 副标题。
media string 媒体元素（图标或图片） - 包含icon或图片的HTML片段。
message string 消息内容。
additionalClass string 添加给通知HTML元素的类，一般用于自定义样式。
closeOnClick boolean false 点击通知的HTML元素时，是否关闭通知。
onClose function 通知关闭的回调函数（无论是手动关闭还是通过调用 $$.closeNotify 方法，都会执行）
