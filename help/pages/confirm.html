$$.confirm(text,[title, callbackOk, callbackCancel],[buttonText])

概述

我们需要调用以下任一一种方法来打开弹层:

$$.confirm(text, [title, callbackOk, callbackCancel])

$$.confirm(text, [title, callbackOk, callbackCancel], [buttonText])

$$.confirm(text, [callbackOk, callbackCancel])

$$.confirm(text, [callbackOk, callbackCancel], [buttonText])

  • text - string 文本
  • title - string 弹层标题，可选。
  • callbackOk - function 当用户点击 “确定” 按钮时，回调函数将被执行，可选。
  • callbackCancel - function 当用户点击 “取消” 按钮时，回调函数将被执行，可选。
  • buttonText - Array 弹层按钮的文本，默认是 “确定” | “取消” ，可以自定义，可选。

实例

弹层:

HTML 代码:
<button type="button" class="Btn BtnPrimary InkRipple ConfirmBtn">确认消息框</button>
js 代码:
$$(".ConfirmBtn").on("click",function(){
  $$.confirm("确认消息框",function(){
    $$.alert("alert警告消息框",function(){
      $$.prompt("欢迎体验KMUI")
    });
  });
});
